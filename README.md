# OpenCV-Object-Detection
![gif1](https://github.com/shinmura0/OpenCV-Object-Detection/blob/master/gif1.gif "gif1")

#Environment
+ Raspberry Pi 3(or PC)
+ USB camera

#Usage
+ Run main_opencv.py.
+ Push [p] to start object detection.
+ Push [q] to end object detection.
